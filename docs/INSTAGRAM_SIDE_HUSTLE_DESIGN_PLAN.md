# インスタ副業ユーザー向け UI デザインプラン

## 現状の課題分析

### 無機質に感じる原因

| 問題点 | 現状 | 影響 |
|--------|------|------|
| **カラーパレットが業務ツール的** | Gray/Indigo/Blue の冷たい配色 | 事務作業のような印象、ワクワク感がない |
| **ナビゲーションが英語** | "Dashboard", "Rankings", "Verification" | ターゲットユーザーに馴染みにくい |
| **ブランド名が技術的** | "Rakuten Rank Check" | ツール感が強く、自分のビジネスを支えてくれる相棒感がない |
| **データ表示が表計算ソフト的** | テーブル主体、数字の羅列 | 「稼げる」実感が湧かない |
| **モチベーション要素がゼロ** | 数値のみの無機質な画面 | 副業で頑張ろうという気持ちが持続しない |
| **モバイル体験が二次的** | デスクトップ優先のレイアウト | インスタユーザーはスマホがメイン |
| **感情に訴えるデザインがない** | フラットで均一なUI | 高料率商品の発見時の喜びが伝わらない |

---

## デザインコンセプト

### テーマ: 「わたしの副業アシスタント」

インスタ副業をしている人が毎日開きたくなる、**親しみやすく、成果が見えて、モチベーションが上がる**ツールへ。

### ペルソナ
- 20-40代、インスタグラムでアフィリエイト副業をしている
- スマホ中心の生活
- デザインや見た目の良さを重視する
- 「今日はいくら稼げそう？」が最大の関心事
- 楽天ROOMやストーリーズで商品紹介をしている

## Phase 1: カラー & ブランディング刷新

### 1-1. カラーパレット変更

**Before (現在)**
```
Primary:    Indigo-600 (#4f46e5) → 業務ツール的
Background: Gray-50 (#f9fafb) → 無機質
Accent:     Blue-600 (#2563eb) → 冷たい
```

**After (提案)**
```css
/* メインカラー: 温かみのあるコーラルピンク〜パープルのグラデーション */
--primary:          #E8567F;  /* コーラルピンク - メインアクション */
--primary-light:    #FFF0F3;  /* 薄ピンク - 背景のアクセント */
--primary-gradient: linear-gradient(135deg, #E8567F 0%, #B847CF 100%); /* インスタ風グラデーション */

/* 背景: 温かみのあるオフホワイト */
--bg-main:     #FDF8F7;  /* ほんのりピンクがかったホワイト */
--bg-card:     #FFFFFF;  /* カード背景 */
--bg-surface:  #FAF5F4;  /* セクション背景 */

/* アクセントカラー */
--accent-gold:    #F5A623;  /* 収益・ポイント表示 */
--accent-mint:    #36D1A0;  /* 成功・上昇表示 */
--accent-coral:   #FF6B6B;  /* 注意・下降表示 */
--accent-purple:  #9B6BFF;  /* 特別料率 */

/* テキスト */
--text-primary:   #2D2D3A;  /* メインテキスト（真っ黒ではなくソフト） */
--text-secondary: #8E8EA0;  /* サブテキスト */
--text-accent:    #E8567F;  /* 強調テキスト */
```

### 3-2. ランキング表示の改善

#### ランキングカードのデザイン改善（モバイル）

```
┌────────────────────────────────────────┐
│  🥇 1位  ↑3  NEW                       │

│  │      │                              │

│                                        │

│  │ 料率 4.0%    │  │ 獲得pt 512pt  │  │
│  │  (特別料率!)   │  │  🎯           │  │
│  └──────────────┘  └────────────────┘  │
└────────────────────────────────────────┘

変更ポイント:

- 特別料率はキラキラ演出（紫のグラデーションバッジ）
- ポイントは金色で強調

- 高料率行: 左ボーダーにアクセントカラー
- ランクバッジ: 1〜3位にグラデーション背景

```

## Before / After イメージ

### 全体の印象
```
Before: 「業務システム」「管理画面」「データベースビューア」
After:  「わたしの副業ツール」「毎日開きたい」「成果が見える」
```

### カラーの印象
```
Before: 寒色系（Gray + Indigo + Blue）→ 冷たい、距離感
After:  暖色系（Pink + Purple + Gold）→ 温かい、親しみ、ワクワク
```

### 情報表示の印象
```
Before: 数字の羅列 → 「で、結局いくら？」
After:  視覚的ハイライト → 「この商品、料率高い！紹介しよう！」
```
